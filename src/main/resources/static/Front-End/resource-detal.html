资源简介:

本套课程主线是Blog项目实战开发。开发过程不采用任何第三方库，通过[纯手工] 编码方式，讲解了 Node.js 的开发流程、知识、思路与开发经验。内容涉及 Node.js 4.x 支持的 ES6语法，并对 Node.js 核心 API 做了介绍。通过这种方式让学员更全面、更透彻的理解 Node.js ，快速入门，直接上手开发，真正找到 Node.js 开发的感觉，扫除深入学习 Node.js 的障碍 。
